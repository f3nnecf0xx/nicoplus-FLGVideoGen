このプログラムはニコニコチャンネルプラス(Sheeta含む)サイトの一部無料の動画から本編を取得するプログラムです。<br>
コードは複雑ですが、やってることは簡単<br>
サイトのファンクラブIDの取得<br>
↓ファンクラブIDをヘッダーに入れる<br>
動画のセッションIDをリクエスト<br>
↓<br>
セッションIDからindex.m3u8を取得<br>
一番上の最高品質のリンクを取得<br>
なぜかここでmediaviewidを引っ張る<br>
↓<br>
最高品質のm3u8を取得した複合キーに上書き<br>
(謎の.key"A"を消去)<br>
完成したデータをディレクトリを作って保存<br>
